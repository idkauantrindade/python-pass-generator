# Python Pass Generator

A **secure and minimal password generator** built with **Python**.  
Generates **strong passwords**, allows you to **add tags**, and stores all passwords in a **JSON file**.  
The system is **CLI-based** and automatically creates the JSON file if it does not exist.

## Features

- Generate **strong passwords** with customizable length  
- Save passwords with **tags/labels**  
- View password history at any time  
- CLI-based interactive menu  
- Automatic JSON file creation  
- Lightweight and easy to extend  

---

## Technologies

- **Python 3.x** – core programming language  
- **JSON** – local storage for passwords  
- **secrets** – secure random password generation  
- **string** – password character sets  
- **datetime** – timestamp each generated password
